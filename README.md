## Quickstart

1\) Clone the repository.

```
git clone https://github.com/mhmh4/minesweeper.git
```

2\) Install npm modules.

```
npm i
```

3\) ...

```
PORT=3000
MONGODB_URI=mongodb://username:password@host:port/database
```

4\) Start the server.

```
npm run dev
```
